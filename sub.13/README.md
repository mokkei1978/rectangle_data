オフライン・ネスティング子モデル用のTIDE側面境界ファイル
========

OFFNESTSUB + TIDE の単体テストで用いる入力ファイル。


データ内容
--------

側面境界ファイル。

  * off_tide_[ens]        東、北、南の潮位、潮流(x,y)
  * off_barotropic_[ens]  東、北、南の非潮汐成分の水位、順圧流量(x,y)
                          (潮汐なし実験の出力と大きくは変わらない)

offnest_tideの設定で作成した。
