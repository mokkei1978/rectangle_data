オフライン・ネスティング子モデル用のGLS側面境界ファイル
========

OFFNESTSUB + GLS の単体テストで用いる入力ファイル。


データ内容
--------

側面境界ファイル。

  * off_gls_[ens]  東、北、南のGLS係数
  * off_vmixcoef_[ens]  東、北、南の鉛直混合係数

offnestpar_glsの設定で作成した。
